# Go Vay AI

Go Vay AI is a web platform for learning, playing, and reviewing Go games with AI-powered explanations.

## Features

- Google/Facebook login
- Beginner Go lessons
- Human vs Human mode
- Human vs Bot mode
- Human vs AI Coach mode
- Game replay
- Post-game analysis
- AI move suggestions
- Vietnamese explanations for mistakes and better moves

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Firebase Authentication
- PostgreSQL / Supabase
- Prisma
- KataGo Engine
- AI Explanation Service
